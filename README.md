# 🤟 ASL Sign Language Detector

Real-time American Sign Language (ASL) alphabet recognition using AI & Computer Vision.

## 📌 About
This project detects ASL hand signs (A-Z) in real-time using a webcam. It uses MediaPipe for hand landmark extraction and a Random Forest Classifier for prediction.

## ✨ Features
- 🎥 Real-time webcam detection
- 🔤 A to Z alphabet recognition
- 💬 Word builder — spell words using hand signs
- 📖 ASL reference chart
- 94.07% model accuracy

## 🛠️ Tech Stack
| Tool | Purpose |
|------|---------|
| MediaPipe | Hand landmark detection |
| OpenCV | Image processing |
| Scikit-learn | Random Forest model |
| Streamlit | Web application |
| Kaggle ASL Dataset | Training data |

## 🚀 How It Works
1. Webcam captures live video
2. MediaPipe extracts 21 hand landmarks
3. Random Forest model predicts the letter
4. Letter displayed on screen in real-time

## 📊 Model Details
- Algorithm: Random Forest Classifier
- Training Samples: 14,000+
- Hand Landmarks: 21 keypoints (x, y, z)
- Accuracy: 94.07%
